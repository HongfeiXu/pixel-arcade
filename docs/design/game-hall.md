# 游戏大厅 — 设计文档

## 概述

游戏大厅是应用首页，展示所有可玩的游戏，孩子点击即可进入。

## 布局

竖屏全屏布局：

```

│  ┌──────────┐  ┌──────────┐  │
│  │ 游戏封面  │  │ 游戏封面  │  │  ← 游戏卡片网格
```

## 游戏卡片

每个卡片包含：

- 游戏封面图（像素风插画或游戏截图）
- 游戏名称
- 历史最高分（星星数）
- 状态标识：可玩 / 即将推出（锁定图标 + 灰色遮罩）

卡片点击后进入对应游戏页面，使用淡入过渡（200ms opacity，参见 [视觉规范](ui-style.md#动画)）。

## 交互

- 可玩的游戏卡片点击后直接进入游戏
- 未上线的游戏卡片点击后轻微抖动，不跳转
- 页面支持纵向滚动（游戏多了之后）
- 底部音效开关控制全局 `soundEnabled` 设置（存入 `pixelarcade_settings`），游戏内的静音按钮读取同一设置

## 数据

- 历史最高分从 `pixelarcade_scores` 读取，无记录时显示"⭐ —"（短横线占位）
- 游戏列表从 `games/registry.ts` 读取
