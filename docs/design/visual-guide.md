# 视觉规范 — 像素风

## 设计原则

- 像素风格，致敬经典游戏机时代
- 色彩鲜明、对比度高，适合幼儿辨识
- 所有元素保持像素网格对齐感
- 圆角最小化，保留像素的方正感

## 配色方案

### 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | `#1A1A2E` | 深蓝黑，护眼 |
| 棋盘背景 | `#16213E` | 略浅于主背景，区分游戏区域 |
| 网格线 | `#1E2A4A` | 微弱可见，辅助定位 |
| 主强调色 | `#FFD600` | 黄色，用于标题、星星、高亮 |
| 次强调色 | `#00E5FF` | 青色，用于辅助信息 |

### 按钮颜色

| 状态 | 颜色 |
|------|------|
| 默认 | `#2D2D5E` 填充 + `#4A4A8A` 描边 |
| 按下 | `#1A1A3E` 填充 + 缩放 95% |

## 字体

使用像素风格的等宽字体。优先级：

1. **Press Start 2P**（Google Fonts，免费像素字体）
2. 系统等宽字体 fallback

```css
font-family: 'Press Start 2P', monospace;
```

> 中文内容暂时使用系统默认字体，像素中文字体资源较少且文件大。

### 字号

| 用途 | 大小 |
|------|------|
| 游戏大厅标题 | 24px |
| 游戏卡片名称 | 14px |
| 游戏内计分 | 16px |
| 按钮文字 | 12px |
| 提示文字 | 10px |

## 动画

所有动画保持像素风的"阶梯感"，避免丝滑的缓动曲线。

| 动画 | 方式 | 说明 |
|------|------|------|
| 方块下落 | 逐格跳跃 | 不做平滑插值，一格一格跳 |
| 消行闪烁 | 透明度 0/1 切换 | 闪烁 3 次，每次 100ms |
| 星星飞行 | steps() 缓动 | CSS `animation-timing-function: steps(8)` |
| 按钮按下 | transform scale | 95% 缩放，50ms |
| 页面切换 | 淡入 | 200ms opacity 过渡 |

## 游戏区域像素渲染

Canvas 渲染需要关闭抗锯齿以保持像素锐利：

```javascript
ctx.imageSmoothingEnabled = false;
```

同时处理 Retina 屏幕：

```javascript
const dpr = window.devicePixelRatio; // iPhone 16 Pro/Max 为 3
canvas.width = logicalWidth * dpr;
canvas.height = logicalHeight * dpr;
canvas.style.width = logicalWidth + 'px';
canvas.style.height = logicalHeight + 'px';
ctx.scale(dpr, dpr);
```

## 覆盖层 / 弹窗样式

游戏中的覆盖层（倒计时、暂停、游戏结束、存档恢复）统一使用以下样式：

| 属性 | 值 |
|------|-----|
| 背景 | `rgba(26, 26, 46, 0.85)`（主背景色 + 85% 不透明度） |
| 文字颜色 | `#FFD600`（主强调色） |
| 按钮样式 | 与虚拟手柄按钮一致（`#2D2D5E` 填充 + `#4A4A8A` 描边） |
| 按钮最小触控区域 | 56 × 56 pt |
| 出现动画 | 无动画，直接显示（保持像素风的干脆感） |

倒计时数字使用 **48px** 像素字体，居中显示。

## 图标

游戏内图标（暂停、静音、星星等）使用像素风格的自绘图标，不引入图标库。用 Canvas 或小尺寸 PNG 精灵图实现。
